# Court Compass

Court Compass is a youth basketball tournament finder for families and players in Iowa, Minnesota, Montana, North Dakota, South Dakota, and Wyoming.

This first prototype includes:

- Tournament search and filters
- Card and map-style views
- Favorites
- Calendar view
- Tournament submission form
- Starter tournament data

Planned next steps:

- Move tournament data into a database
- Connect Apify for scheduled tournament discovery
- Add admin approval for submitted and scraped tournaments
- Deploy the app online
